# Dataset Files

Place the HPD dataset CSV files in this directory:

1. `1- the-number-of-cases-of-infectious-diseases.csv`
2. `2- the-worlds-number-of-vaccinated-one-year-olds.csv`
3. (Optional) `4- excess-deaths-cumulative-economist-single-entity.csv`
4. (Optional) `5- number-of-reported-cholera-deaths.csv`

## Download Sources

Download from Kaggle: [Pandemics in World Dataset](https://www.kaggle.com/datasets/imtkaggleteam/pandemics-in-world)
